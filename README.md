# Калькулятор пожаров на промышленном предприятии

Предварительные требования
- Java Development Kit (JDK) 17 или выше
- Apache Maven 3.8 или выше

Установка и запуск
```
git clone https://github.com/Melancholia312/fires-calculator.git
```
```
cd fires-calculator
```
```
mvn clean install
```
```
mvn spring-boot:run
```
Доступ к приложению
```
http://localhost:8081/
```
